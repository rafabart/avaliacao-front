Projeto avaliação frontend. CRUD de Agendamento de cursos e Categorias.

Usando:

* React
* Axios
* Primereact
* Bootswatch


Para rodar o projeto, apos baixar do git rodar os comando na pasta raiz do projeto:

```npm install```

```npm start```


URL de acesso:
